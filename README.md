## Getting Started

1. Create a new project based on my template

```
npx create-next-app --example https://github.com/SibaldH/mystack [PROJECT_NAME]
```

2. run the server:

```
cd [PROJECT_NAME]
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000).
