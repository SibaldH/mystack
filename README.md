## Getting Started

First clone the project.

````
git clone https://github.com/SibaldH/mystack.git
```

Then install the dependencies:

````

cd mystack-2024
npm i

```

Then run the server:

```

npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
```
